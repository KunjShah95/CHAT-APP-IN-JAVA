## ğŸ“‹ Installation

### Prerequisites

- Java 11 or higher
- Git

### Steps

1. Clone the repository
    ```sh
    git clone https://github.com/yourusername/java-chat-app.git
    ```

2. Navigate to the project directory
    ```sh
    cd java-chat-app
    ```

3. Compile the application
    ```sh
    javac *.java
    ```

4. Run the server
    ```sh
    java ChatServer
    ```

5. Run the client application
    ```sh
    java ChatClient
    ```
